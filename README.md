### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `Navicat`

### 三、功能描述
系统分为（身份）：普通用户、员工、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 个人中心（修改个人信息、修改密码、修改头像）
4. 收货地址
5. 药品收藏历史
6. 我的药品订单
7. 订单退款
8. 首页
9. 公告
10. 留言板
11. 药品一览
12. 药物详情
13. 我的购物车
14. 人工在线客服

**员工功能：**
1. 登录
2. 个人中心（修改密码、修改个人信息、修改头像）
3. 客服聊天在线回复
4. 公告信息查看
5. 留言板一览
6. 药物药品一览
7. 药品评价一览
8. 药品订单一览
9. 药品入库一览

**管理员功能：**
1. 登录
2. 个人中心
3. 管理员管理
4. 客服聊天管理
5. 基础数据管理（公告类型、药品类型、药品入库类型）
6. 公告管理
7. 留言板管理
8. 药品管理
9. 药品评价管理
10. 药品收藏管理
11. 药品订单管理
12. 药品入库管理
13. 用户管理
14. 员工管理
15. 轮播图管理
